# 디자인 명세서 (Design Specification)

본 명세서는 "포슬카페 사장님 대시보드" 웹 애플리케이션의 색상 체계, 레이아웃 그리드, 타이포그래피 계층, 컴포넌트 스펙 및 마이크로 모션 인터랙션 가이드를 담고 있습니다.

---

## 🎨 1. 컬러 시스템 (Color Palette)

커피 전문점의 아이덴티티를 살리기 위해 **따뜻한 에스프레소 브라운(Mocha & Espresso Brown)**과 **부드러운 크림 화이트(Cream White)**를 기본 톤앤매너로 삼고, 활력을 주기 위한 **테라코타 오렌지**를 포인트 컬러로 설정했습니다.

| 레이어명 | 컬러 칩 | HEX Code | 사용처 및 디자인 가이드 |
| :--- | :--- | :--- | :--- |
| **Cream Sand (Background)** | ▨ | `#FDFCF7` | 모바일 디바이스 프레임 내부 기본 배경색 (매우 밝은 크림색) |
| **Coffee Cream (Card BG)** | ▨ | `#F5F0E6` | 주요 카드 테두리 및 서브 컴포넌트 배경색 |
| **Muted Sand (Border)** | ▨ | `#EADFC9` | 카드 경계선, 스크롤바 트랙, 기본 분할 가이드라인 |
| **Mocha Brown (Graph / Info)** | ▨ | `#8C6F56` | 실시간 매출 그래프 메인 선, 서브 포인트 텍스트 |
| **Espresso Brown (Main Text)** | ▨ | `#4E3629` | 대형 타이틀, 본문 텍스트, 메인 아이콘, 탭 바 비활성 색상 |
| **Point Orange (Action)** | ▨ | `#C25E35` | "발주" 버튼 및 모달 확인 등의 핵심 액션 유도 버튼색 (테라코타 오렌지) |
| **Trend Green BG (Status)** | ▨ | `#EBF4E0` | 매출 상승을 시각화하는 변동 배지의 밝은 연두색 배경 |
| **Trend Green Text (Status)** | ▨ | `#4E7D3A` | 매출 상승 배지 내부 텍스트 및 그래프 피크(Peak) 활성 노드색 |

## 📐 2. 그리드 & 레이아웃 (Layout & Grids)

데스크톱 브라우저 중앙에 실물 크기의 스마트폰을 배치하여 실감 나는 모바일 경험을 제공합니다.

* **디바이스 목업 크기 (Viewport Size)**
  * **너비 (Width)**: `420px` 고정 (최대 모바일 폭 대응)
  * **높이 (Height)**: `850px` 고정
  * **코너 반경 (Border Radius)**: `50px` (둥근 곡선 액정 연출)
  * **액정 테두리 (Border)**: `8px border-stone-300`
* **내부 간격 (Spacing System)**
  * **기본 안쪽 여백 (Global Padding)**: `px-5` (`20px` 좌우 마진)
  * **컴포넌트 간 세로 마진 (Vertical Gap)**: `space-y-5` (`20px` 간격 배정)
  * **그리드 간격 (Grid Gap)**: `gap-2.5` (`10px` 조밀한 간격 적용)

## ✍️ 3. 타이포그래피 계층 (Typography Hierarchy)

한국어 가독성을 해치지 않도록 시스템 산세리프 폰트를 사용하여, 모던하고 깔끔한 산디과적 그리드를 유지합니다.

| 계층 (Hierarchy) | 크기 (Size) | 굵기 (Weight) | 사용처 | 적용 클래스 (Tailwind) |
| :--- | :--- | :--- | :--- | :--- |
| **L1. 대형 강조 헤더** | `20px` (1.25rem) | Black (`900`) | 상단 사장님 환영 타이틀 | `text-xl font-black` |
| **L2. 실시간 숫자 금액** | `30px` (1.875rem) | Black (`900`) | 매출 금액 수치 강조 | `text-3xl font-black` |
| **L3. 카드 내부 값** | `16px` (1rem) | Bold (`700`) | 판매 잔수, 객단가 수치값 | `text-base font-bold` |
| **L4. 주요 알림 타이틀** | `12px` (0.75rem) | Bold (`700`) | 할 일 목록 제목, 모달 소제목 | `text-xs font-bold` |
| **L5. 캡션 & 서브 정보** | `10px` (0.625rem) | Medium (`500`) | 명언 설명, 시간대, 변동율 배지 | `text-[10px] font-medium` |

## 🧩 4. 주요 컴포넌트 상세 스펙 (Component Specs)

### ① 상단 웰컴 블록 (Mascot Welcome Header)
* **스펙**: 에스프레소 브라운 배경 (`bg-coffee-800`, `#4E3629`)의 큰 곡선 헤더 블록. 하단에만 곡률(`rounded-b-[40px]`)을 주어 액정 상단을 아늑하게 감싸는 형태입니다.
* **캐릭터 에셋**: 3D 그래픽 느낌을 재현한 둥근 라이트 베이지 원형 무대 위에, 손 흔드는 이모지 `🙋‍♂️`와 따뜻한 모락모락 커피잔 `☕`를 3D 입체 카드처럼 결합.
* **모션 가이드**: 캐릭터 전체 레이어에 위아래로 둥실 뜨는 부드러운 애니메이션(`animate-bounce` 2.5초 주기)을 주어 재미 요소를 추가했습니다.

### ② 실시간 매출 카드 (Real-time Sales Card)
* **스펙**: 밝은 화이트 샌드 카드 (`bg-white`)에 아주 연한 모카 브라운 테두리 (`border-coffee-200/50`) 처리.
* **SVG 라인 차트**:
  * 곡선 패스 코드: `M 10 90 C 60 85, 90 88, 130 80 C 170 72, 210 75, 250 62 L 290 50`
  * 채우기 그라데이션: `#C3B091`에서 흰색으로 서서히 투명해지는 그라데이션 마스크 적용.
  * 종착 노드 점: `Trend Green Text` 색상의 노드 포인트에 퍼지는 펄스 모션 (`animate-pulse`)을 주어 실시간 연동 중임을 인지시킵니다.

### ③ 신속 안전 발주 모달 (Quick Order Modal)
* **스펙**: 흰색 배경의 바텀 시트(Bottom Sheet) 모달로, 화면 하단에서 슬라이드 업 형태로 출력됩니다.
* **딤드 배경 (Dimmed Layer)**: `bg-black/40 backdrop-blur-xs`를 주어 팝업 시 뒷배경 흐림 처리를 제공합니다.
* **완료 피드백**: '즉시 발주하기' 실행 시, 할 일 목록의 우유 재고 항목이 자동으로 비활성 처리(`opacity-40` 및 가로선 관통효과)되며 확인 배지로 치환됩니다.
